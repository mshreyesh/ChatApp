# ChatApp
This is a chatting app where in users can chat with one another by sending text messages. New users can Register by using email and password, and existing members can login. If the user forgets his/her password it can be reset too,by clicking on forgot password and typing the email registered the user with, the user will get an email on that email Id for password resetting instructions. Also users can set Profile pictures with photos and images. Also the app shows online and offline status of other users.
